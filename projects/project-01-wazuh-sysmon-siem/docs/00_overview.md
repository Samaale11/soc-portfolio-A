# Overview

This project builds a SOC-style pipeline: Windows telemetry (Sysmon + Windows Event Logs) → Wazuh SIEM → detections → investigations → tickets.

## Deliverables (definition of done)
- Wazuh running (dashboard proof)
- Windows agent connected (proof)
- Sysmon installed and generating events (proof)
- Sysmon events visible inside Wazuh (proof)
- 2 custom detections firing (proof)
- 2 investigation tickets written with evidence and decision
